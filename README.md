# Excel_Sales_Dashboard
Interactive Excel Sales Dashboard Project
Tools used:
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
  Objective:
  - To analyse sales performance and customer purchasing behavior using an Interactive Excel Dashboard.
    Key Features:
    - KPI summary
    - Sales trend analysis
    - Customer Segmentation
    - Interactive slicers
      Outcome:
      - Created a clean, professional dashboard suitable for business decision making and reporting
